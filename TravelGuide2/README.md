# Туристический путеводитель

## Запуск
```bash
cd TravelGuide
dotnet run
```
Открой браузер: http://localhost:5000

---

## Как добавить фотографии

Все фото кладутся в папку `wwwroot/photos/`.
Ожидаемые имена файлов (можно использовать .jpg или .png):

### Города
| Файл                  | Что это              |
|-----------------------|----------------------|
| moscow.jpg            | Фото Москвы          |
| moscow-coat.png       | Герб Москвы          |
| spb.jpg               | Фото Санкт-Петербурга|
| spb-coat.png          | Герб Санкт-Петербурга|
| kazan.jpg             | Фото Казани          |
| kazan-coat.png        | Герб Казани          |

### Достопримечательности
| Файл                  | Что это                  |
|-----------------------|--------------------------|
| red-square.jpg        | Красная площадь          |
| tretyakov.jpg         | Третьяковская галерея    |
| kremlin.jpg           | Московский Кремль        |
| hermitage.jpg         | Эрмитаж                  |
| peterhof.jpg          | Петергоф                 |
| kazan-kremlin.jpg     | Казанский Кремль         |
| kul-sharif.jpg        | Мечеть Кул-Шариф         |

> Если файл не найден — карточка просто покажется без картинки, ошибки не будет.

## ВАЖНО — перед первым запуском

Если ты уже запускал предыдущую версию проекта, удали старый файл базы данных:
```
bin/Debug/net8.0/travelguide.db
```
Иначе старые пути к фото останутся в базе.
